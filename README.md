# 漫画名言データベース

キャラクター名・作品名・キーワードであいまい検索できる漫画名言サイトです。  
有志の皆さんのコントリビュートでデータを増やしています。

## 名言を投稿する方法

### 方法1：Airtableフォーム（簡単・オススメ）

👉 **[名言投稿フォーム](YOUR_AIRTABLE_FORM_URL)**

フォームに以下を入力して送信するだけです。管理者が確認後、次回の自動更新（毎時）で反映されます。

| 項目 | 内容 |
|------|------|
| 名言 | セリフをそのまま |
| キャラクター | キャラクター名 |
| 作品名 | 漫画・アニメのタイトル |
| 場面 | どのシーンで言われたか（1〜2文） |
| 意味・教訓 | この名言が伝えること（1〜2文） |
| タグ | カンマ区切りで複数可（例：努力,友情,バトル） |

### 方法2：Pull Request（エンジニア向け）

`data/quotes.json` に直接追記してPRを送ることもできます。

```json
{
  "quote": "名言テキスト",
  "chara": "キャラクター名",
  "series": "作品名",
  "scene": "どんな場面で言われたか（1〜2文）",
  "meaning": "この名言の意味・教訓（1〜2文）",
  "tags": ["タグ1", "タグ2"]
}
```

## セットアップ（サイト管理者向け）

### 1. Airtableの準備

1. [Airtable](https://airtable.com) でベースを作成
2. テーブル名：`名言データ`
3. 以下のフィールドを作成：

| フィールド名 | 種類 | 備考 |
|------------|------|------|
| 名言 | 単一行テキスト | |
| キャラクター | 単一行テキスト | |
| 作品名 | 単一行テキスト | |
| 場面 | 長文テキスト | |
| 意味・教訓 | 長文テキスト | |
| タグ | 複数選択 | |
| ステータス | 単一選択 | 選択肢：「審査中」「公開済み」|

4. 投稿フォームを作成し、URLを `index.html` の `YOUR_AIRTABLE_FORM_URL` に設定

### 2. GitHub Secrets の設定

GitHubリポジトリの **Settings → Secrets and variables → Actions** に以下を追加：

| シークレット名 | 値 |
|--------------|-----|
| `AIRTABLE_API_KEY` | AirtableのPersonal Access Token |
| `AIRTABLE_BASE_ID` | ベースID（URLの `appXXXXXX` 部分） |
| `AIRTABLE_TABLE_NAME` | `名言データ` |

### 3. GitHub Pages の有効化

**Settings → Pages → Branch: main / folder: / (root)** で有効化

### 4. 動作確認

Actionsタブから `Airtable から名言データを同期` を手動実行して `data/quotes.json` が更新されればOK。

## 仕組み

```
有志 → Airtableフォームで投稿
     ↓
管理者がステータスを「公開済み」に変更
     ↓
GitHub Actions（毎時）がAirtable APIからデータ取得
     ↓
data/quotes.json を自動コミット
     ↓
GitHub Pages がサイトを自動更新
```
